# ADR-009: フロントエンドライブラリの選定（ルーティング・CSS・UIコンポーネント）

| 項目 | 値 |
|------|-----|
| 日付 | 2026-02-22 |
| ステータス | 採用 |

## コンテキスト

React アプリのルーティング、CSS フレームワーク、UI コンポーネントライブラリを選定する。

## 決定

| カテゴリ | 採用 |
|---------|------|
| ルーティング | React Router |
| CSS フレームワーク | Tailwind CSS |
| UI コンポーネント | shadcn/ui（Radix UI ベース） |

## 理由

- **React Router**: 定番ライブラリで情報量が多い。ページパス（`/`, `/podcast/:id`, `/settings`）を自然に扱える
- **Tailwind CSS**: ユーティリティファーストで素早くスタイリング可能。Tauri アプリとの相性も良い
- **shadcn/ui**: コピー＆ペースト方式のコンポーネント集。ダイアログ、ボタン、テーブル、バッジなど本アプリで必要な UI 部品が揃っている。依存が少なくカスタマイズしやすい
